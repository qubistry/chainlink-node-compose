# Docker Compose

Uses a basic `docker-compose` setup to launch 5 chainlink nodes and a single postgres DB.

## Node Access

| Node Number | Address               |
| ----------- | --------------------- |
| 1           | http://localhost:6711 |
| 2           | http://localhost:6722 |
| 3           | http://localhost:6733 |
| 4           | http://localhost:6744 |
| 5           | http://localhost:6755 |
